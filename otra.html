¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado que usamos para estructurar contenido en la web. Las páginas web se componen de elementos (etiquetas) que describen títulos, párrafos, imágenes, enlaces, listas y más.

Estructura básica de un documento HTML

Ejemplo mínimo de un archivo HTML:

<!DOCTYPE html>
<html lang="es">
	<head>
		<meta charset="utf-8" />
		<title>Mi página</title>
	</head>
	<body>
		<h1>Hola Mundo</h1>
		<p>Mi primer sitio web.</p>
	</body>
</html>

Consejo: guarda el archivo con extensión .html y ábrelo en tu navegador para ver el resultado.

Títulos y párrafos

Usa <h1>...</h1> a <h6> para títulos y <p> para párrafos.

<h2>Subtítulo</h2>
<p>Este es un párrafo.</p>

Listas

Listas ordenadas y desordenadas:

<ul>
	<li>Elemento 1</li>
	<li>Elemento 2</li>
</ul>

<ol>
	<li>Primero</li>
	<li>Segundo</li>
</ol>

Enlaces e imágenes

Enlaces con <a href="..."> e imágenes con <img src="..." alt="..."/>. 

<a href="https://example.com" target="_blank">Visitar ejemplo</a>

<img src="imagen.jpg" alt="Descripción de la imagen" width="300"/>

Tablas

Tablas simples con encabezado y filas:

<table>
	<thead>
		<tr><th>Nombre</th><th>Edad</th></tr>
	</thead>
	<tbody>
		<tr><td>Ana</td><td>28</td></tr>
	</tbody>
</table>

Formularios

Un ejemplo de formulario simple:

<form action="/enviar" method="post">
	<label>Nombre:<input type="text" name="nombre" /></label>
	<button type="submit">Enviar</button>
</form>

Nota: para probar envío real necesitas un servidor. Puedes usar mailto: o herramientas locales para simular.

Elementos semánticos

HTML5 incluye etiquetas semánticas como <header>, <nav>, <main>, <article>, <section> y <footer>. Úsalas para mejorar la accesibilidad y SEO.

Comentarios

Los comentarios no se muestran en el navegador:

<!-- Este es un comentario en HTML -->

Atajos

  • <!DOCTYPE html> — declara la versión de HTML
  • <meta charset="utf-8"> — codificación recomendada
  • <img alt="..."> — siempre incluir texto alternativo
